## Install

Clone this repo:  
`$ git clone https://github.com/corleo/pycodes.git ~/pycodes/`  

Add following line to `~/.bash_aliases` or `~/.bashrc` script to automatically source `.pycodesrc` file:  
`[ -f ~/pycodes/.pycodesrc ] && source ~/pycodes/.pycodesrc`
